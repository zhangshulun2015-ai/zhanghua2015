# 图文视频知识库 v4.0 中文专享版（张华制作）使用说明

这是一个面向“人查资料 + AI 投喂”的本地知识库。它会把视频、图文帖、本地文档、图片和手动笔记整理成 Markdown，并生成本地 Dashboard。

## 核心入口

```bash
python scripts/configure_vault.py "<知识库根路径>"
python scripts/migrate_v3.py --vault "<知识库根路径>"
python scripts/regenerate_dashboard.py --vault "<知识库根路径>"
```

## 自动识别与新建知识库

首次使用时，AI助手会确认知识库根目录。如果目录中还没有 `图文视频知识库`，导出脚本会自动新建：

- `wiki/` 笔记目录
- `raw/视频文件/` 原始视频目录
- `raw/图片文件/` 图片素材目录
- `raw/文档文件/` PDF、Word、表格、PPT、txt/md 原文件目录
- `WIKI-SCHEMA.md`
- `wiki/index.md`
- `wiki/log.md`
- 10 个分类目录

如果你给的是 `图文视频知识库` 子目录，脚本会自动归一到包含 `index.html` 的知识库根目录。

知识库路径示例：

```text
D:\我的知识库
```

Dashboard 位于：

```text
<知识库根路径>\index.html
```

## 本地视频入库

```bash
python scripts/transcribe.py "<本地视频路径>" -o "<输出目录>"
python scripts/export_to_obsidian.py "<transcript.txt>" --vault "<知识库根路径>" --analysis analysis.json --video "<本地视频路径>"
python scripts/regenerate_dashboard.py --vault "<知识库根路径>"
```

导出会把视频复制到 `图文视频知识库\raw\视频文件\`。Dashboard 中的“删除本地视频”只删除视频文件，不删除笔记。

## 网络视频入库

```bash
python scripts/download_video.py <URL>
python scripts/transcribe.py <视频路径> -o <输出目录>
python scripts/export_to_obsidian.py <transcript.txt> --vault <知识库根路径> --analysis analysis.json --video <视频路径>
python scripts/regenerate_dashboard.py --vault <知识库根路径>
```

支持抖音、B站、小红书、X。

自动流程是：识别平台和链接 -> 抓取/下载视频 -> ffmpeg 转码或提取音频 -> 语音转文字 -> AI 提取标题、分类、标签、摘要、知识点、金句和 AI快速参考 -> 导出 Markdown 笔记 -> 复制素材到 raw -> 刷新 Dashboard。

导出脚本默认会原样保留转录文本，适合已经做过 AI 语义校对或人工整理的稿子。只有需要把原始单行稿重新自动断句时，才额外加 `--format-transcript`。长视频默认在笔记中展示 AI 语义整理稿，并挂上完整纯文本字幕、原始 VTT/原始分段稿链接；只有明确要求全文展示时，才把逐字完整版直接写进笔记正文。

平台验证结论：

- 小红书：公开视频不用登录，直接转发/分享链接可下载视频或提取图文；Chrome Cookie 被占用时会自动无 Cookie 重试。
- B站：公开视频不用登录，直接复制网页链接可下载；Chrome Cookie 被占用时会自动无 Cookie 重试。
- 抖音：已验证可用；遇到登录、滑块或风控验证时，需要先在浏览器完成验证。
- X：已验证可用；公开内容可尝试直接提取，登录可见内容需要 Cookie。X 帖子可能同时带缩略图和长视频，入库前要先检查 `duration`、`formats`、`subtitles` 等元数据；只要检测到视频，就走视频下载、转录和总结流程，不按纯图文处理。

## 手动添加笔记

```bash
python scripts/add_note.py --vault <知识库根路径> --title "标题" --category "AI教程资料" --summary "一句话摘要" --content "核心内容"
```

也可以在 Dashboard 右上角点“添加笔记”。

## 本地文件入库

```bash
python scripts/ingest_document.py "<本地文件路径>" --vault "<知识库根路径>" --ocr auto
```

支持：

- PDF：文字 PDF 直接提取；扫描 PDF 在 OCR 依赖可用时自动识别。
- Word：`.docx`。
- 表格：`.xlsx`、`.csv`。
- PPT：`.pptx`，老 `.ppt` 建议先另存为 `.pptx`。
- 图片：`.jpg/.png/.webp/.bmp/.tif`，可选 OCR。
- 文本：`.txt/.md`。

提取层命令：

```bash
python scripts/extract_document.py "<本地文件路径>" --ocr auto
```

入库脚本会复制原文件到 `图文视频知识库\raw\文档文件\`，图片复制到 `raw\图片文件\`，然后生成标准笔记并刷新 Dashboard。

## 换电脑 OCR 安装提醒

如果要在别的电脑上识别扫描 PDF 或图片文字，需要额外安装 OCR 环境。普通文字 PDF、Word、Excel/CSV、PPT、txt/md 不受影响。

推荐安装项：

```bash
pip install pytesseract pdf2image pillow
winget install --id tesseract-ocr.tesseract
winget install --id oschwartz10612.Poppler
```

中文识别还需要 `chi_sim.traineddata`。本机为了避开中文用户名路径问题，使用：

```text
C:\Tesseract-OCR\tessdata
C:\Poppler\bin
```

换电脑后如果 OCR 不生效，先检查 Tesseract、中文语言包 `chi_sim`、Poppler 和 `pytesseract` 是否都已安装。

## AI 快速参考

每篇 v4.0 笔记都会包含：

```markdown
## 🤖 AI快速参考

- 适合回答的问题：
- 可直接引用的结论：
- 关键术语：
- 适用场景：
- 关联资料：
```

投喂 AI 时优先复制 AI 快速参考，不默认复制完整转录；长视频需要深读时，使用单篇资料里的“全文投喂AI”，会把完整字幕/转录、路径、摘要和 AI快速参考一起复制。

## Dashboard 能力

- 10 个目录栏筛选。
- 卡片、列表、AI参考三种视图。
- 复制当前筛选给 AI。
- 按目录投喂。
- 按主题投喂。
- 单篇笔记复制给 AI。
- 长视频全文投喂 AI。
- 本地文件入库命令生成。
- 回收站恢复。
- 本地视频空间提醒和删除。
